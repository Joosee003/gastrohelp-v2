# GastroHelp V2

Sitio comercial premium y panel demo interactivo para GastroHelp.

## Archivos principales
- `index.html`: web comercial.
- `demo-panel.html`: entorno demo interactivo.
- `styles.css`: sistema visual y responsive.
- `app.js`: interacciones de la web.
- `demo.js`: navegación y datos ficticios del panel demo.

No requiere build ni dependencias. Puede publicarse en GitHub Pages, Netlify, Vercel o cualquier hosting estático.
